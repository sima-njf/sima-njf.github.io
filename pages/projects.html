Academic Projects

  • epiworldRcalibrate — R package for ML-based calibration of epidemic agent-based models (three-layer bidirectional LSTM calibrator; accuracy/runtime gains over ABC). Status: in development.
  • imaginarycss — R package for Cognitive Social Structures (CSS-native data model, perceptual error taxonomy, tie-level accuracy, accuracy-preserving null models). Status: in development.
  • Comparative evaluation of Rt estimators — Agent-based network models to assess performance of effective reproduction number estimators; collaboration with CDC/SDBC.
  • Machine Generalize Learning for ABM calibration — Supervised calibrator beyond surrogate models; integrated with epiworldR for practical workflows.
  • EHR/VA workflows — Reproducible pipelines for large clinical datasets (QA, versioning, and privacy-aware analysis).
  • Neurotransmitter classification via AFTDS + ML — LSTM on time-dependent fluorescence with UV plasmonic enhancement; probe-free detection of DA/NE/DOPAC.
